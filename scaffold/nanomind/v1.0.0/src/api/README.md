# API

This directory is reserved for auto-generated API clients.

Use `gve api generate` to generate TypeScript clients from Thrift IDL definitions.

**Do not manually edit files in this directory.**

## Usage

1. Define your API contracts in Thrift IDL files under the project's `api/` directory
2. Run `gve api generate` to generate TypeScript clients
3. Import and use the generated clients in your views
